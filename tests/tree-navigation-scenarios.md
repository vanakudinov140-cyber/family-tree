# Ручные сценарии: навигация и режимы дерева

Playwright не обязателен. Проверять на текущей семье.

## Режимы видимости

1. **Ближайшие** (`nearby`) — только ближайший круг (супруги, родители, дети, братья/сёстры + их супруги, сородители детей). Без бабушек и внуков.
2. **3 поколения** (`generations`) — родители, focus (+ супруги и братья/сёстры), дети. Без прародителей и внуков.
3. **Вся ветка** (`branch`) — все предки и все потомки без ограничения глубины + супруги на пути + братья/сёстры focus.
4. **Вся семья** (`all`) — все люди.

## Четыре поколения (регресс)

Цепочка: Татьяна Васильевна Бурнашева → Михаил Алексеевич Кудинов → Иван Михайлович Кудинов → Демид Иванович Кудинов.

1. Выбрать Татьяну, режим **Вся ветка** — видны все четыре поколения (Татьяна, Михаил, Иван, Демид).  
2. Супруги Михаила/Ивана на одном уровне с партнёром, не «воруют» корень ветки.  
3. Валентин/Анна Носковы → Надежда → Иван → Демид — тоже четыре поколения в branch.  
4. Focus на Иване в branch — предки (родители и бабушки) и потомок Демид.  
5. Focus на Демиде в branch — все известные предки по линиям Ивана и Дины.  
6. Focus на старшем — все потомки без обрезания на 3 уровне.

## 3 поколения vs ветка

7. Focus Иван, **3 поколения** — родители и Демид есть; Татьяна (прабабушка относительно Демида / бабушка Ивана) не показывается как лишний уровень выше родителей Ивана… Для Ивана бабушка Татьяна = −2 и **не** входит в generations.  
8. Тот же focus, **Вся ветка** — Татьяна снова видна.

## Базовые

9. Поиск Ивана — focus+select, центрирование, подсветка.  
10. nearby остаётся локальным.  
11. all показывает всех.  
12. Сворачивание скрывает потомков, но не focused и путь предков к нему.  
13. Выбор человека в скрытой ветке временно раскрывает путь.  
14. «Развернуть все» возвращает все поколения.  
15. URL `person` + `view` (`nearby` | `generations` | `branch` | `all`).  
16. heritage / diagram не ломаются при смене view mode.  
17. Focused person не исчезает.

## Unit

Запуск без браузера:

```bash
npx tsx tests/tree-visibility.selftest.ts
```

Проверяет: 4 поколения в branch/layout, compact generations, nearby, 8 искусственных поколений, цикл родителей без зависания.
