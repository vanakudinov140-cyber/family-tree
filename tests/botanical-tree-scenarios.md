# Botanical tree — visual verification checklist

## Modes

- [ ] **Дерево → Ближайшие** — одно крупное дерево, фокусный человек удобно в кадре, родители ниже, дети выше
- [ ] **Дерево → 3 поколения** — три ясных «яруса», дерево занимает 65–85% высоты, не антенна
- [ ] **Дерево → Вся ветка** — весь род фокуса, боковые роды супругов свёрнуты (+N)
- [ ] **Дерево → Вся семья** — одно большое дерево или «сад» из нескольких компонентов, без микроскопических точек

## Focus centers

- [ ] Иван как центр — его линия основная, линия супруги вторым стволом
- [ ] Дина как центр — зеркально
- [ ] Демид как центр — отец слева снизу, мать справа снизу, потомки выше

## Edge cases

- [ ] Человек без супруга — один медальон на ветви
- [ ] Несколько браков — отдельные боковые ответвления
- [ ] Открытый профиль — камера смещается, layout не пересчитывается
- [ ] Desktop и mobile — pan/zoom, клик по медальону и +N

## Must NOT appear in Дерево

- [ ] Нет стрелок и пунктирных relationship edges
- [ ] Нет union/junction nodes
- [ ] Нет подписей «I уровень», «II уровень»

## Схема

- [ ] Точные parent/spouse связи сохранены
- [ ] Карточки с фото, плавные линии, читаемые типы связей
