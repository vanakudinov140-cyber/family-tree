# Heritage visual mode — manual test scenarios

Проверка режимов «Дерево» (`heritage`) и «Схема» (`diagram`).
Данные, Supabase, SQL и состав видимых людей не должны меняться.

## Подготовка

1. Открыть приложение с загруженным деревом (~41 человек).
2. По умолчанию активен режим **Дерево**.
3. Dev-сервер не останавливать.

---

## Визуальная доводка (регресс)

1. **Вся семья** не уменьшает людей до нечитаемых точек — zoom ≥ `ALL_OVERVIEW_MIN_ZOOM` (~0.45).  
2. Все люди остаются на canvas (pan по краям допустим).  
3. Focused Татьяна Васильевна сразу заметна (двойное кольцо / «Центр»).  
4. Прямая линия Татьяна → Михаил → Иван → Демид выделяется.  
5. Остальные линии приглушены, но видны.  
6. При zoom in появляются детали карточек (compact → full).  
7. При zoom out детали скрываются без изменения layout / hit-box.  
8. Большой пустой белый контейнер слева отсутствует — только компактные floating-панели.  
9. Toolbar не закрывает дерево.  
10. Ветви заметны на молочном фоне.  
11. В режиме **Вся ветка** карточки крупнее и читаемее.  
12. В **Ближайшие** — полный вид карточек.  
13. **Схема** визуально как раньше (без heritage-эффектов).  
14. Minimap показывает узлы и focused person; можно скрыть.  
15. «Показать центр» возвращает focused с читаемым zoom.  
16. Панель профиля и admin-кнопки работают.  
17. Сворачивание веток работает.  
18. Поиск работает.  
19. Realtime не сбрасывает visualMode / focus.  
20. При ~41 человеке нет заметных тормозов.

## Переключение режимов

- Дерево ↔ Схема сохраняет focus, selected, viewMode, collapsed.  
- nearby / generations / branch / all не меняют состав людей при смене visual mode.

## Пересечения карточек (регресс)

1. «Вся семья» — нет ни одной наложенной карточки.  
2. «Вся ветка» Ксении — Роман и дети не перекрываются.  
3. «Вся ветка» Егора — Виктория и дети не перекрываются.  
4. «Вся ветка» Ивана — братья, сестра и супруги читаются.  
5. «Вся ветка» Татьяны — четыре поколения без пересечений.  
6. Zoom out — карточки не начинают накладываться (координаты фиксированы).  
7. Zoom in — плашки не выходят за hit-box `200×178`.  
8. overview/compact/full не двигают узлы.  
9. Focused ring не увеличивает карточку (без scale).  
10. Search highlight не увеличивает карточку.  
11. Сворачивание/разворачивание — новый корректный layout.  
12. Diagram mode размеры `230×125` без изменений.
